EU Railway Analysis Platform

Decision-support tool for sustainable mobility in Europe

Context

Passionate about rail mobility and committed to ecological transition, I developed an analysis platform for European train punctuality data using open data from national operators.

Problem

Railway performance data is scattered across many sources and formats. It's difficult for citizens and decision-makers to objectively compare European networks.

Solution

Development of a data processing pipeline: automated extraction via APIs (SNCF, DB, NS, SNCB, etc.), cleaning and normalization with Pandas, interactive visualizations, deployment on GitHub Pages (temporary hosting for demonstration).

Tech stack: Python 3, Pandas, Requests, Plotly, GitHub Pages.

Impact

  • Analysis of 50,000+ trips over 6 months
  • Identification of recurring delay patterns
  • Tool shared with Young European Federalists (JEF)

Technologies

Python 3 Pandas Plotly REST APIs
