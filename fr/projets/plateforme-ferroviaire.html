Plateforme d'analyse ferroviaire UE

Outil d'aide à la décision pour la mobilité durable en Europe

Contexte

Passionné par la mobilité ferroviaire et engagé dans la transition écologique, j'ai développé une plateforme d'analyse des données de ponctualité des trains en Europe, en utilisant les données ouvertes des opérateurs nationaux.

Problème

Les données de performance ferroviaire sont éparpillées entre de nombreuses sources et formats différents. Il est difficile pour les citoyens et décideurs de comparer objectivement les réseaux européens.

Solution

Développement d'un pipeline de traitement de données :

  • Extraction automatisée via APIs (SNCF, DB, NS, SNCB, etc.)
  • Nettoyage et normalisation avec Pandas
  • Visualisations interactives (graphiques temporels, cartes)
  • Déploiement sur GitHub Pages (hébergement temporaire pour démonstration)

Stack technique : Python 3, Pandas, Requests, Plotly, GitHub Pages.

Impact

  • Analyse de 50 000+ trajets sur 6 mois
  • Identifications de patterns de retard récurrents
  • Outil partagé avec l'association Young European Federalists (JEF)
  • Contribué à un rapport sur la mobilité transfrontalière

Technologies

Python 3 Pandas Plotly APIs REST GitHub Pages

Apprentissages

  • Manipulation de grands volumes de données
  • Intégration d'APIs multiples
  • Visualisation de données complexes
  • Déploiement statique (temporaire)

Note : Projet en cours, déploiement GitHub Pages temporairement actif pour démonstration.
